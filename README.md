# Low-Code-Cyber-Threat-Intelligence-Dashboard
Developed a low-code Cyber Threat Intelligence Dashboard using Microsoft Power BI by integrating AbuseIPDB and AlienVault OTX APIs to provide real-time visibility into threat trends, risk prioritization, and data freshness.

## Overview
This project demonstrates how threat intelligence data can be transformed into actionable insights using Microsoft Power BI.

## Data Sources
- AbuseIPDB API
- AlienVault OTX API

## Features
- Threat trends and patterns
- Risk level classification
- Feed freshness monitoring
- Interactive filtering

## Tools Used
- Power BI
- REST APIs
- Data modelling (DAX)

## Outcome
Improved visibility of threat intelligence data and supported faster security decision-making.
